# 🔓 QSUnlocker

QSUnlocker — это графический инструмент для Windows, предназначенный для восстановления системы, снятия ограничений, управления процессами и выполнения системных операций.

Программа разработана на Python с использованием Tkinter и предоставляет удобный интерфейс для администрирования и устранения проблем в системе.

---

## ⚠️ Важно

> Данный инструмент работает с системными настройками Windows (реестр, процессы, службы).
>
> ❗ Требуются права администратора  
> ❗ Используйте на свой страх и риск  
> ❗ Рекомендуется только для опытных пользователей  

---

## ✨ Возможности

### 🧩 Основное
- Перезапуск Explorer
- Открытие диспетчера задач
- Встроенная CMD-консоль

### 🖱️ Доступность
- Отключение `SwapMouseButton`
- Управление `sethc.exe` (Sticky Keys exploit / восстановление)

### 🔐 Безопасность
- Снятие ограничений системы:
  - Task Manager
  - Registry Editor
  - CMD
  - Control Panel
  - Run (Win + R)
- Очистка IFEO Debugger (перехваты процессов)

### ⚙️ Система
- Очистка временных файлов
- Сброс сети (Winsock / IP)
- Перезапуск DNS
- Проверка системы:
  - `chkdsk`
  - `sfc /scannow`
  - `DISM`

### 🔌 Питание
- Перезагрузка / выключение / гибернация

### 🔥 Безопасность системы
- Сброс правил брандмауэра
- Включение / отключение UAC

### 📊 Мониторинг процессов
- Просмотр всех процессов
- Определение:
  - 🔴 Критичных (системных)
  - 🟠 Подозрительных
- Завершение процессов
- Отображение CPU

### 🚀 Автозагрузка
- Просмотр автозапуска (HKCU / HKLM)
- Удаление записей

### 🧱 Shell восстановление
- Проверка текущего shell
- Сброс на `explorer.exe`

### ⚙️ Настройки
- Редактирование `struct.json` прямо в приложении
- Настройка:
  - тем
  - процессов
  - реестра
  - UI
